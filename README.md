# ghub

## Setup

- Go installed
- VSCode installed
- VSCode Go extension
- Go packages
```bash
go get -u github.com/golang/protobuf/protoc-gen-go
go get -u github.com/golang/protobuf/proto
```
- Protoc compiler
```bash
$ brew install protobuf
```
Verify installation:
```bash
$ protoc --version
libprotoc 3.11.4
```
